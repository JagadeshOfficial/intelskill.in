# **App Name**: LearnFlow

## Core Features:

- Public Forum: Allows guest users to engage with discussion boards; incorporates AI moderation tools for content.
- Content Display: Allows users to watch and download material
- Online Compiler: Enables students to practice coding in multiple languages, integrated directly within the learning platform.
- Themed UI: Allow users to switch between various theme options such as dark mode and high contrast mode

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to convey calmness and trust.
- Background color: Light, desaturated blue-gray (#F0F8FF) for a clean and unobtrusive backdrop.
- Accent color: Muted coral (#F08080) to draw attention to interactive elements without overwhelming the user.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-focused feel.
- Body font: 'Inter' (sans-serif) provides excellent readability for longer text blocks.
- Use minimalist, line-based icons to maintain a clean and modern aesthetic.
- Responsive layout to ensure seamless usability across devices. Prioritize clear information hierarchy and intuitive navigation.
- Subtle transitions and micro-interactions to provide feedback and enhance user engagement without distracting from the learning experience.
